# FoxPRO-HR

FoxPRO-HR is a web-based HR management system scaffold built with Django and designed for deployment on PythonAnywhere.

## Included in this repository
- Complete software specification: `SOFTWARE_SPECIFICATION.md`
- Step-by-step development guide: `DEVELOPMENT_GUIDE.md`
- Modular Django app structure covering all requested HR modules
- Modern responsive UI with transitions and animation

## Quick start
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```

Then open:
- `/` → Dashboard
- `/specification/` → Full system specification
- `/implementation-guide/` → Build walkthrough
